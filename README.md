# Calculator (en cours)

Réalisé en : 
- HTML
- SCSS
- JavaScript

Utilisation de **dart-sass** pour compiler mon **.scss** en **.css**

Pour la compilation, j'effectue la commande suivante :
- `./dart-sass/sass assets/sass/style.scss assets/css/style.css --watch`
